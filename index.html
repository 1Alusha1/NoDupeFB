Привет

Приложение предназначено для автоматического блокирования дублирующихся аккаунтов на Facebook. Пожалуйста, следуйте приведённым инструкциям для успешного использования.

О приложении

Это приложение позволяет автоматически блокировать дублирующие аккаунты на Facebook. Для работы необходимо заполнить форму:

  • Тип работы приложения — выберите один из двух вариантов:
    • По ID — блокировка выполняется по числовым ID пользователей.
    • По имени — блокировка выполняется по имени и фамилии пользователя.
  • Список пользователей — введите в текстовое поле список ID или имён пользователей, в зависимости от выбранного типа работы. Каждый ID или имя указывайте с новой строки (разделитель — перенос строки).

После ввода всех данных и запуска приложения авторизуйтесь, выберите нужный аккаунт и через 2 минуты начнётся блокировка указанных пользователей.
