Привет

Приложение предназначено для автоматического блокирования дублирующихся аккаунтов на Facebook. Пожалуйста, следуйте приведённым инструкциям для успешного использования.

О приложении

Это приложение позволяет автоматически блокировать дублирующие аккаунты на Facebook. Для работы необходимо заполнить форму из трёх полей:

  • Логин и пароль — данные для авторизации бота, чтобы он мог выполнять блокировку от имени нужного аккаунта;
  • Список ID пользователей — вводится в текстовое поле (textarea) через запятую. В список можно добавлять как числовые ID, так и ссылки, содержащие ID. Для корректного распознавания ID используется регулярное выражение: /(\d{8,})/.

После ввода всех данных и запуска, бот авторизуется и приступит к блокировке указанных пользователей.
